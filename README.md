Workplace Strategy Analytics: Data-Driven Insights README
Overview
This project aims to modernize workplace strategy by centralizing attendance and leave data to better understand employee behavior. By analyzing these patterns, the organization moves away from anecdotal policy-making toward a model that optimizes real estate footprints and supports employee health. Ultimately, the project transforms raw behavioral data into actionable insights that balance operational efficiency with a sustainable, wellness-focused culture. 
________________________________________
Dataset
The analysis is based on internal organizational records, including:
•	Attendance Data: Daily office check-ins and remote vs. on-site logs. 
•	Leave Data: Leave types, frequency, and duration across different departments. 
•	Employee Metadata: Sector, industry role, and tenure information. 
________________________________________
Tools & Technologies
•	Power BI: Core visualization and interactive dashboarding. 
•	Power Query: ETL (Extract, Transform, Load) for data cleaning. 
•	DAX (Data Analysis Expressions): Used for advanced calculations and KPIs. 
•	Google Gemini: Utilized for synthesizing dashboard findings into professional executive summaries and slide deck content. 
•	Microsoft PowerPoint: Final presentation of actionable insights. 
________________________________________
Steps & Implementation
1.	Data Loading & Power Query: Connected to raw data sources and used Power Query to perform data profiling, handle missing values, and standardize formats. 
2.	Data Modeling: Created relationships between tables and developed custom DAX measures for metrics like Average Weekly Occupancy and Leave Correlation Rates. 
3.	Dashboard Design: Built an intuitive interface with interactive slicers, focusing on Operational Efficiency and Sustainable Culture. 
4.	AI Reporting: Used Google Gemini to analyze dashboard takeaways and generate professional content for a PowerPoint deck. 
________________________________________
Dashboard Highlights
•	Heat Maps: Visualizing peak office attendance days to optimize utilities and space. 
•	Trend Lines: Tracking leave patterns to identify seasonal burnout risks. 
•	Actionable KPIs: Indicating specific areas where real estate costs can be reduced without impacting productivity. 
________________________________________
Results
•	Eliminated 100% of reliance on anecdotal evidence for hybrid work policies. 
•	Identified specific real estate consolidation opportunities based on actual peak occupancy. 
•	Created a repeatable framework for monitoring employee health through behavioral metrics.

How to Run
1.	Install Power BI Desktop: Ensure you have the latest version installed. 
2.	Load Data: Open the .pbix file and update the data source settings to point to your local dataset files. 
3.	Refresh: Click the Refresh button to populate the visuals with the cleaned data. 
4.	View Report: Interact with the dashboard slicers to filter by department or time period
