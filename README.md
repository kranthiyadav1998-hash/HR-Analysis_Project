Project Overview
This project aims to modernize workplace strategy by centralizing attendance and leave
data to better understand employee behavior. By analyzing these patterns, the
organization moves away from anecdotal policy-making toward a data-driven model that
optimizes real estate footprints and supports employee health. It transforms raw

behavioral data into actionable insights, balancing operational efficiency with a wellness-
focused culture.

Dataset

The analysis is based on internal organizational records, including:
Attendance Data: Daily office check-ins, remote vs. on-site logs.
Leave Data: Leave types, frequency, and duration across different departments.
Employee Metadata: Sector, industry role, and tenure information.

Tools & Technologies
Power BI: Core visualization and interactive dashboarding.
Power Query: ETL (Extract, Transform, Load) for data cleaning.
DAX (Data Analysis Expressions): For advanced calculations and KPIs.
Google Gemini: Utilized for synthesizing dashboard findings into professional
executive summaries and slide deck content.
Microsoft PowerPoint: Final presentation of actionable insights.

Step-by-Step Implementation

1. Data Loading & Power Query ETL
Connected to raw data sources and used Power Query to perform data profiling.
Handled missing values, standardized date formats, and removed duplicates to
ensure data integrity.

2. Data Modeling & DAX Calculations
Created relationships between tables and developed custom measures for metrics
such as Average Weekly Occupancy, Leave Correlation Rates, and Year-over-Year
Performance Fluctuations.

3. Dashboard & Report Development
Designed an intuitive interface with interactive slicers. Focused on two primary views:
Operational Efficiency (Space usage) and Sustainable Culture (Wellness/Health
indicators).

4. AI Synthesis & Presentation
Leveraged Google Gemini to analyze the dashboard's key takeaways and generate
professional-grade content for a PowerPoint deck targeted at industry leaders.

Dashboard Highlights
Heat Maps: Visualizing peak office attendance days to optimize utilities and space.
Trend Lines: Tracking leave patterns to identify seasonal burnout risks.
Actionable Insights: KPIs indicating exactly where real estate costs can be reduced
without impacting productivity.

Project Results
Eliminated 100% of reliance on anecdotal evidence for hybrid work policies.
Identified specific real estate consolidation opportunities based on actual peak
occupancy.
Created a repeatable framework for monitoring employee health through behavioral
metrics.

How to Run

Install Power BI Desktop: Ensure you have the latest version installed.
Load Data: Open the .pbix file and update the data source settings to point to
your local dataset files.
Refresh: Click the Refresh button to populate the visuals with cleaned data.
View Report: Interact with the dashboard slicers to filter by department or time
period.
